memoria flash: o que seria?

serve para armazenamento de arquivos (fotos, documentos, etc).

tabela de referencia: memoria

ruim ok bom otimo
128 GB 256 GB 516 GB 1 TB

uma foto de celular, um documento, tem algumas dezenas de MB.

  • 1000 MB = 1 GB
  • 1000 GB = 1TB

pontos importantes

  • evitar hdd (mais devagar)
  • procurar por ssd (mais rapido)

ram

  • random access memory
  • deixa o computador mais rapido (responsivo)

recomendacao

o dell inspiron eh a pior das opcoes.

a principal diferenca entre as opcoes da vaio esta na ram, na escolha de sistema operacional, e no preco.

sistema operacional

linux eh melhor que windos em termos de atualizacoes, seguranca contra virus, e eficiencia.

por conta da maior eficiencia do sistema operacional linux, a opcao de laptop com esse sistema pode ser mais rapida ate mesmo que o windows de 16 GB. mesmo se houver uma forte preferencia para com os 16 GB do windows, o website da vaio tem uma opcao do fe16 por 3.199,00 com linux (continua mais barato que a opcao windows).

16 GB eh uma quantidade suficiente para uso basico da web e de aplicativos de office.
